# Hello World Languages

A beginner-friendly collection of simple "Hello, World!" programs written in multiple popular programming languages.

## Included Languages

- Python
- JavaScript
- Java
- C
- C++
- C#
- Go
- Rust
- PHP

## Files

| Language | File |
|---|---|
| Python | hello.py |
| JavaScript | hello.js |
| Java | Hello.java |
| C | hello.c |
| C++ | hello.cpp |
| C# | hello.cs |
| Go | hello.go |
| Rust | hello.rs |
| PHP | hello.php |

## How to Run

### Python
```bash
python hello.py
```

### JavaScript
```bash
node hello.js
```

### Java
```bash
javac Hello.java
java Hello
```

### C
```bash
gcc hello.c -o hello
./hello
```

### C++
```bash
g++ hello.cpp -o hello
./hello
```

### C#
```bash
csc hello.cs
hello.exe
```

### Go
```bash
go run hello.go
```

### Rust
```bash
rustc hello.rs
./hello
```

### PHP
```bash
php hello.php
```
